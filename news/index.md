# Changelog

## pvEBayes 0.3.0

CRAN release: 2026-06-16

\*Passed rOpenSci software peer review with gold grade

\*Improved the documentation and vignette.

\*Included two opioid-related datasets

\*Expanded and improved the unit tests.

\*Fixed bugs.

\*Improved message output.

## pvEBayes 0.2.2

CRAN release: 2026-03-13

\*Updated the CVXR interface for compatibility with CVXR (\>= 1.8.1).

## pvEBayes 0.2.1

CRAN release: 2026-01-11

\*Removed the dependency on ‘SobolSequence’.

## pvEBayes 0.2.0

CRAN release: 2025-12-15

- Added a new implementation of the Koenker-Mizera (KM) model using the
  ‘CVXR’ package.
- Removed the dependency on ‘REBayes’.
- Removed the suggested dependency on ‘Rmosek’.
- Added support for supplying a prespecified expected‐null baseline
  estimator via the E argument in ‘pvEBayes()’.
- Added a package vignette.
- Added a CITATION file, enabling use of citation(“pvEBayes”).

## pvEBayes 0.1.2

CRAN release: 2025-10-01

- fixed one bug that was causing eyeplot testing example to fail.
- Added a `NEWS.md` file to track changes to the package.
